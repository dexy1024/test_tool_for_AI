# 图像识别系统测试平台

这是一个基于 Vue 3 + Element Plus 的图像识别系统后台测试功能页面，用于测试图像识别准确率。

## 功能特性

### 1. 场景选择
- 端架 (end_shelf)
- 货架 (shelf) 
- 挂架 (hanging_rack)
- 落地架 (floor_stand)
- 展示架 (display_stand)
- 促销架 (promotion_stand)

### 2. 规则编码选择
- 规则编码001 - 商品摆放检测
- 规则编码002 - 价格标签检测
- 规则编码003 - 商品完整性检测
- 规则编码004 - 陈列规范检测
- 规则编码005 - 清洁度检测

### 3. 输出项编码选择
- 输出项001 - 基础识别结果
- 输出项002 - 详细分析结果
- 输出项003 - 质量评估结果
- 输出项004 - 合规性检查结果

### 4. 批量图片上传
- 支持拖拽上传
- 支持多文件选择
- 支持 jpg/png 格式
- 单个文件最大 10MB
- 实时上传进度显示

### 5. 回调地址配置
- 支持 URL 格式验证
- 用于接收识别结果回调

### 6. 识别结果展示
- **图片缩略图**: 点击可放大查看
- **是否合格**: 合格/不合格状态标签
- **原因说明**: 详细的合格或不合格原因
- **SKU名称**: 识别出的商品名称
- **数量**: 识别出的商品数量

### 7. 图片查看功能
- 点击缩略图可放大查看原图
- 支持图片详情信息展示
- 响应式设计，适配移动端

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Element Plus** - Vue 3 组件库
- **Vite** - 快速构建工具
- **Axios** - HTTP 客户端

## 安装和运行

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 构建生产版本
```bash
npm run build
```

### 4. 预览生产版本
```bash
npm run preview
```

## 使用说明

1. **选择场景**: 从下拉菜单中选择测试场景
2. **选择规则编码**: 选择相应的识别规则
3. **选择输出项编码**: 选择输出结果类型
4. **上传图片**: 拖拽或点击上传测试图片
5. **配置回调地址**: 输入接收结果的回调URL
6. **开始识别**: 点击按钮开始图像识别测试
7. **查看结果**: 在结果表格中查看识别结果

## 注意事项

- 当前版本使用模拟数据进行演示
- 实际使用时需要对接真实的图像识别API
- 图片上传功能需要配置真实的上传接口
- 回调地址需要实现相应的接收逻辑
